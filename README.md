# FluentCart Invisible CAPTCHA Pro (v3.0)

An advanced security plugin for WordPress that integrates Google reCAPTCHA v3 seamlessly into FluentCart and Fluent Forms. It protects your checkout from bots without adding friction for real human users.

* **Real-time Toaster Alerts**: Beautiful SweetAlert2 notifications for blocked attempts.
